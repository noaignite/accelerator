### `lerp`

Linear interpolation between two numbers.

- @param `start` - The start value.
- @param `end` - The end value.
- @param `amount` - The amount to interpolate between the two values.
- @returns The interpolated value.

#### Example

```ts
lerp(0, 100, 0.5) // 50
lerp(0, 100, 0.75) // 75
```
