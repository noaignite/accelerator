# Accelerator AI Skills

Agent skills for common NoA Ignite workflows.

## Available Skills

### `utils`

General-purpose TypeScript utilities used across apps and packages.

### `react-utils`

Shared React hooks and component helpers.

### `types`

Shared TypeScript helper types.

### `style-guide`

Shared ESLint, Prettier, and TypeScript config presets.

### `centra-types`

TypeScript definitions for Centra API data structures.

### `centra-mocks`

Mock Centra API responses for tests and local development.

### `react-centra-checkout`

React helpers and UI building blocks for Centra checkout flows.

### `tailwind-typography`

Tailwind plugin for responsive typography and fluid values.

### `react-native-eslint`

Opinionated ESLint config for React Native and TypeScript projects.

### `create-app`

CLI for scaffolding NoA Ignite templates inside a Turborepo project.

## Installation

```bash
npx skills add noaignite/accelerator
```

## Skill Structure

Each skill contains:

- `SKILL.md` - instructions for the agent
- optional supporting `.md` files for larger skills
