# @noaignite/react-native-eslint

## Purpose

Opinionated ESLint config for React Native + TypeScript projects.

## Reference docs

- Installed package: `node_modules/@noaignite/react-native-eslint/dist/docs/`
- Repo-local fallback: `packages/react-native-eslint/dist/docs/`

## Install

```bash
npm i -D @noaignite/react-native-eslint
```

## Use when

- You are setting up linting for a React Native app
- You need a shared RN ESLint baseline

## Rules

- Use for React Native projects only.
- Prefer this package over assembling a custom RN ESLint stack from scratch.
