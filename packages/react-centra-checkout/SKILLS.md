# @noaignite/react-centra-checkout

## Purpose

React helpers and UI building blocks for Centra checkout flows.

## Install

```bash
npm i @noaignite/react-centra-checkout
```

## Public API

See `src/index.ts` in source or `dist/index.js` in the published package for the full surface.

## Use when

- You are building or testing Centra checkout UI
- You need shared checkout state or integration helpers

## Rules

- Prefer this package for checkout-related React behavior.
- Use it together with `@noaignite/centra-types` and `@noaignite/utils` when needed.
