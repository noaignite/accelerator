# @noaignite/centra-types

## Purpose

TypeScript definitions for Centra API data structures.

## Install

```bash
npm i @noaignite/centra-types
```

## Public API

See `src/index.ts` for the full surface.

## Use when

- You are modeling Centra requests or responses
- You need shared contracts for Centra integrations

## Rules

- Prefer these types over handwritten Centra DTOs.
- Keep Centra-specific logic aligned with these definitions.
