# @noaignite/utils

Utils - Utility functions.

## Installation

Utils is available as an [npm package](https://www.npmjs.com/package/@noaignite/utils).

```sh
// with pnpm
pnpm add @noaignite/utils

// with yarn
yarn add @noaignite/utils

// with npm
npm install @noaignite/utils
```
