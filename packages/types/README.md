# @noaignite/types

TypeScript utilities. These types help keep common patterns consistent, reduce duplication, and make package APIs easier to work with.

## Installation

`@noaignite/types` is available as an [npm package](https://www.npmjs.com/package/@noaignite/types).

```sh
// with pnpm
pnpm add @noaignite/types

// with yarn
yarn add @noaignite/types

// with npm
npm install @noaignite/types
```
