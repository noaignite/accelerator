# @noaignite/types

Types - Type helpers by NoA Ignite.

## Installation

Types is available as an [npm package](https://www.npmjs.com/package/@noaignite/types).

```sh
// with pnpm
pnpm add @noaignite/types

// with yarn
yarn add @noaignite/types

// with npm
npm install @noaignite/types
```
