# @noaignite/tailwind-typography

## Purpose

Tailwind plugin for responsive typography and fluid values.

## Install

```bash
npm i @noaignite/tailwind-typography
```

## Public API

See `src/index.ts` in source or `dist/index.js` in the published package for the full surface.

## Use when

- You are configuring Tailwind typography scales
- You want consistent responsive text sizing

## Rules

- Prefer the plugin over custom typography math.
- Use in Tailwind projects only.
